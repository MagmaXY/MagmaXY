## 1. Математика

^^

### 1.1 Многочлены и неравенства

^^

#### 1.1.1 Теория

* **Определение** **: Многочлен — это выражение вида**$P(x)=a_{n}x^{n}+a_{n-1}x^{n-1}+\cdot\cdot\cdot+a_{1}x+a_{0}$^^^^^^^^.
* **Формулы Виета** : Для уравнения **$x^{2}+px+q=0$** с корнями **$x_1, x_2$**:
* **$x_1 + x_2 = -p$**^^
* **$x_1 x_2 = q$**^^
* **Симметрические выражения** :
* **Сумма квадратов:**$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2$^^.
* **Сумма кубов:**$x_1^3 + x_2^3 = (x_1 + x_2)^3 - 3x_1x_2(x_1 + x_2)$^^.
* **Сумма коэффициентов** : Равна значению многочлена при **$x=1$**, т.е. **$P(1)$**^^^^^^^^.

#### 1.1.2 Разбор задач прошлых лет

 **Задача (2024)** **: Найти **$\frac{1}{x_1} + \frac{1}{x_2}$** для**$x^2 - 12x + 7 = 0$^^.

* **Решение** : По Виету **$x_1+x_2=12$**, **$x_1x_2=7$**. **Итого:**$12/7$^^.

 **Задача (2025)** **: При каком **$a$** сумма корней **$3x^2 - 9ax + a = 0$** равна 9?**^^.

* **Решение** : Сумма корней **$= 3a$**. **$3a = 9 \Rightarrow a = 3$**^^.

### 1.2 Арифметика остатков и прогрессии

^^

#### 1.2.1 Теория

* **Сравнение по модулю** **: **$a \equiv b \pmod m$**, если **$(a - b)$** делится на**$m$^^.
* **Циклы последних цифр** **: Степени чисел 2, 3, 7 имеют цикл длины 4, степень 9 — длину 2**^^.
* **Прогрессии** :
* **Сумма первых **$n$** нечетных чисел:**$1 + 3 + ... + (2n-1) = n^2$^^.
* **Сумма арифметической прогрессии:**$S = \frac{n(a_1 + a_n)}{2}$^^.

#### 1.2.2 Разбор задач прошлых лет

 **Задача** **: Найти последнюю цифру**$1 + 3 + 5 + ... + 2013$^^.

* **Решение** : Это сумма **$n=1007$** нечетных чисел. **$1007^2$** заканчивается на 9 (**$7^2=49$**)^^.

 **Задача** **: Найти**$gcd(10^{27}-1, 10^{18}-1)$^^.

* **Решение** : **$gcd(a^m-1, a^n-1) = a^{gcd(m,n)}-1$**. **$gcd(27,18)=9$**. **Ответ:**$10^9-1$^^^^^^^^.

### 3.1 Основы и разбор задач

**Задача: Расчет времени работы**

* Робот-пылесос (0.5 м/с), два двигателя по 30 Вт, электроника (24 В, 2 А), аккумулятор 150 Вт·ч.
* **Решение** :

1. Мощность двигателей: **$2 \cdot 30 = 60$** Вт.
2. Мощность электроники: **$P = U \cdot I = 24 \cdot 2 = 48$** Вт.
3. Общая мощность: **$60 + 48 = 108$** Вт.
4. Время: **$150 / 108 \approx 1.39$** ч.

**Задача: Сложение цветов RGB**

* Включены красный и зеленый каналы.
* **Решение** : Красный + Зеленый = Желтый.
